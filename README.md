# Telegram Order Bot V2

Бот для приёма заказов на торты и десерты с полной автоматизацией и мониторингом.

## Возможности

### Для пользователей
- 🎂 Заказ тортов (по весу, от 2 кг)
- 🧁 Заказ десертов (порционно)
- 📋 Каталог из 42 позиций (29 тортов, 13 десертов)
- 🎨 Выбор оформления (крем, шоколад, ягоды, мастика)
- ✍️ Надпись на торте
- 🎭 Дополнения: топпер (+200₽), свеча-фейерверк (+150₽), фотопечать (бесплатно)
- 📍 Два пункта выдачи: Тольятти 11, Циолковского 36
- 🔒 Соблюдение 152-ФЗ (согласие на обработку персональных данных)

### Для владельца
- `/stats` - статистика заказов (сегодня, неделя, месяц, топ-5)
- `/health` - состояние бота и базы данных
- `/backup_now` - ручное создание бэкапа
- `/help` - справка для пользователей
- `/support` - контакты для связи

### Автоматизация
- ⏰ Автоматические бэкапы каждый час (хранятся 24 последних)
- 🔍 Мониторинг с Telegram-уведомлениями (каждый час)
- 📊 Утренние отчёты в 8:00 МСК
- 🧹 Автоочистка застрявших состояний и старых черновиков

## База данных

### Таблицы
- `users` - пользователи с согласиями на обработку данных
- `orders` - заказы со всеми полями (тип, вкус, вес, оформление, дополнения)
- `catalog_cakes` - каталог тортов и десертов
- `conversation_states` - состояния диалогов

### Каталог
- **29 тортов** (цена за кг: 2750₽, праздничные 1.5кг: 4500₽)
- **13 десертов** (порционные: 450₽, специальные: 630-2750₽)
- **Napoleon**: 3 позиции (порция 450₽, торт 2750₽/кг, праздничный 4500₽)

## Развёртывание

### Docker
```bash
cd /root/telegram_bot_v2
docker build -t telegram-order-bot-v2 .
docker run -d --name telegram-order-bot-v2 \
  --restart unless-stopped \
  -v /root/telegram_bot_v2/data:/app/data \
  telegram-order-bot-v2
```

### Cron задачи
```cron
# Автоматические бэкапы (каждый час)
0 * * * * /usr/bin/python3 /root/backup_database_v2.py

# Мониторинг (каждый час в :15)
15 * * * * /usr/bin/python3 /root/monitor_bot_v2.py

# Утренние отчёты (8:00 МСК = 5:00 UTC)
0 5 * * * /usr/bin/python3 /root/send_morning_report_v2.py >> /root/report_v2.log 2>&1
```

## Мониторинг

### Логи
- Бот: `docker logs telegram-order-bot-v2`
- Бэкапы: `/root/db_backup_v2.log`
- Мониторинг: `/root/bot_monitoring_v2.log`
- Отчёты: `/root/report_v2.log`

### Бэкапы
- Расположение: `/mnt/data/telegram_bot/backups_v2/`
- Формат: `orders_v2_YYYYMMDD_HHMMSS.db`
- Хранение: последние 24 бэкапа (24 часа)

### Алерты
Мониторинг автоматически отправляет уведомления владельцу при:
- Проблемах с контейнером
- Застрявших состояниях диалогов (>24ч)
- Старых черновиках заказов (>48ч)
- Ошибках в логах
